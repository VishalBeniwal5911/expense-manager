# Expense Management REST API

A secure RESTful backend API for managing personal expenses with JWT authentication, CRUD operations, modular routing, and MongoDB integration.

---

## Features

- JWT Authentication
- Secure Protected Routes
- CRUD Operations
- Expense Tracking
- Middleware Validation
- Centralized Error Handling
- Modular Backend Architecture
- MongoDB Integration
- RESTful API Design

---

## Tech Stack

### Backend
- Node.js
- Express.js
- JWT
- bcrypt

### Database
- MongoDB
- Mongoose

### Tools
- Postman
- Git & GitHub
- Render
